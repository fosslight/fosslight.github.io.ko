## What is FOSSLight?

<img src="../assets/img/fosslight_system.jpg" alt="FOSSLight System"><br>
FOSSLight은 Open Source와 License를 관리하고, OSC(Open Source Compliance) Process를 순차적으로 처리할 수 있는 통합 시스템입니다.
